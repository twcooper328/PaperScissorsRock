Read Me file for Paper Scissors Rock

What is this:
This is a simple text based rock paper scissors game.
This is a practice project. 

How to play:
manouver past the main menu to the betting screen. 
Adjust you bet by entering "more" or "less".
move quickly through the menus by entering "p".
select either rock, paper, or scissors.
The game is over when you run out of coins.
